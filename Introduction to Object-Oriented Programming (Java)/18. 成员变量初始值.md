### 1. 变量使用核心规则
* **先声明，后赋值**：Java 中所有的变量必须先声明，后赋值才能使用。
* **成员变量自动初始化**：Java 中的**成员变量**（在类中定义，但在方法之外的变量），在创建对象的时候都会执行一次初始化操作，系统都会自动给它们分配一个**默认值**。

### 2. 数据类型的默认值

| 数据类型分类 | 默认值 | 备注说明 |
| :--- | :--- | :--- |
| **基本数据类型** | `0` / `false` | 数值型（如 `int`, `double`）默认值为 `0` 或 `0.0`；<br>`boolean` 类型的默认值是 `false`。 |
| **引用数据类型** | `null` | 包括类、接口、数组等类型。 |

### 3. 关于 `null` 的理解
* `null` 表示**空**，意味着什么都没有。
* 它在内存中主要起到一个**占位**的作用，表示该引用变量目前没有指向任何实际的对象。
