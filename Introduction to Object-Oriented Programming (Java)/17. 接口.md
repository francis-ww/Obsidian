接口实际上是一种特殊的抽象类
接口中所有的方法都是抽象方法
接口使用interface来声明

类只能单继承，接口支持多实现

接口同样具有多样性
接口可以把很多不相关的内容进行整合

特点：
1. 接口中的所有方法都是抽象方法，都是公开的
2. 接口中所有的变量都是quan'ju'jin

```
/**
 *   能继承接口的只能是接口
 *   接口和类只能是实现关系 implements
 */
 
public interface Valuable {
	public abstract void getMoney();
}
```

```
public interface Protection {
	public void protect();
}
```

```
public class Animal {
	public void eat(){
		System.out.println("Animals eat");
	}
}
```

```
public class Gold implements Valuable {
	//必须要实现抽象方法
	public void getMoney(){
		System.out.println("Gold is valuable");
	}
}
```

```
public class Panda extends Animal implements Protection, Valuable {
	public void eat() {
		System.out.println("Panda eats bamboo");
	}
	public void protect() {
		System.out.println("Panda needs protection");
	}
	public void getMoney() {
		System.out.println("Panda is valuable");
	}
}
```
