this：当前类的对象
this可以在方法内部获取到对象的属性信息
this还可以区分局部变量和成员变量


```
public class Car(){
    String color;
    int speed;
    int capacity = 5;
    
    public void run(){
        System.out.println("车能跑");
    }
    
    public static void main(String[] args){
        Car c = new Car;
        c.color = "红色";
        c.speed = 120;
        
        c.run(); //在调用方法时，java会自动地把对象传递给方法，在方法中由thislai'jie
        
    }
}
```
