super表示父类中的内容
this表示自己类中的内容
用super和this来区分父类和子类重名的内容

```
public class Hero() {
	String name = "英雄";
}
```

```
public class WuKong extends Hero {
	String name = "孙大圣";
	public void eat(){
		//this先找自己类，再找父类
		System.out.println(this.name+"在吃桃子");
		System.out.println(super.name+"在吃桃子");
	}
	public static void main(String[] args){
		WuKong s = new WuKong();
		s.eat();
	} 
}
```
