super表示父类中的内容
this表示自己类中的内容
用super和this来区分父类和子类重名的内容

```
public class Hero {
	String name = "英雄";
	
	public Hero(String name){
		System.out.println("我是父类的构造");
		this.name = name;
	}
}
```

```
public class WuKong extends Hero {
	String name = "孙大圣";
	public WuKong(){
		super("武大郎"); //还原程序，在子类构造方法第一行，默认调用父类的构造方法
		System.out.println("我是子类的构造方法");
	}
	public void eat(){
		//this先找自己类，再找父类
		System.out.println(this.name+"在吃桃子");
		System.out.println(super.name+"在吃桃子");
	}
	public static void main(String[] args){
		WuKong s = new WuKong();
		s.eat();
	} 
}
```
创建对象过程：先父类，再子类
