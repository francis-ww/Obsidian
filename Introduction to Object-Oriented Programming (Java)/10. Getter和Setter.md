成员变量一般使用private来声明，保护成员变量不被胡乱的赋值

setter主要是给成员变量赋值，做一定的保护
getter是从成员变量中获取数据

```
package k1;

public class Person {
//    String name;
//    int age;
    private String name;
    private int age;
    
    public void setName(String name){
	    this.name = name;
    }
    
    public void eat(){
        System.ou.println(this.name+"在吃东西");
    }
}
```
