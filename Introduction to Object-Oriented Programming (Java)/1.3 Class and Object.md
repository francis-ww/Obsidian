## e.g. 造车
1. 画图纸
    1. 定义车属性：color, speed, capacity...
    2. 定义车的功能：running...
2. 拿图纸找工厂生产

## 面向对象
 - 类：图纸
    属性：这一类事物的共同属性
    功能：这一类事物共同执行的功能
- 对象：使用类创建的具体的某个东西

- 对象能干什么 - 完全取决于类是如何定义的

Coding
    类要用class来定义
    属性：成员变量来描述，直接写在类中的变量
    功能：成员方法，不写static

public class Car {
    //成员变量
    String color;
    int speed;
    int capacity;
}
