## e.g. 造车
1. 画图纸
    1. 定义车属性：color, speed, capacity...
    2. 定义车的功能：running...
2. 拿图纸找工厂生产

## 面向对象
 - 类：图纸
    属性：这一类事物的共同属性
    功能：这一类事物共同执行的功能
- 对象：使用类创建的具体的某个东西

- 对象能干什么 - 完全取决于类是如何定义的

Coding
    类要用class来定义
    属性：成员变量来描述，直接写在类中的变量
    功能：成员方法，不写static

e.g.
public class Car {
    // 成员变量
    String color; // 颜色
    int speed; // 速度
    int capacity = 5; // 座位
    
    // 成员方法
    public void run(){
        System.out.println("车能跑");
    }
    
    public static void main(String[] args) {
        //int a = 10; // 写在方法里的变量，局部变量
        // 创建对象
        // int a = 10;
        // String color = "呵呵"
        
        // 在面向对象的世界里，变量是没有市场的，这种变量被称为引用。
        // java分为两种数据类型：1. 基本数据类型  2. 引用数据类型  String和我们创建的所有形如的类
        Car c = new Car(); // 创建对象  创建了一辆车，后面想用这两车，就需要使用c来访问
        
        // 让车去跑.
        
        // 对象或者引用.方法()
        c.run(); // . 表示调用，"的"
        
        c.color = "green";
        c.speed = 120;
        
        System.out.println(c.color);
        
        Car c2 = new Car();
        c2.color = "red";
        c2.speed = 180;
        
        System.out.println(c2.color);
        System.out.println(c2.speed);
    }
}
